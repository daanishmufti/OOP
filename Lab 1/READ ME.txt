## Dynamic Kitchen Management System

### Description

This project is a Dynamic Kitchen Management System implemented in C++. It allows users to manage orders, calculate bills, mark orders as completed, and view pending orders.

### Features

- Add new orders with various food items and quantities.
- Print bills for specific orders.
- Mark orders as completed.
- Check for pending orders.
- Display order details by order number.

### Usage

1. Compile the program using a C++ compiler such as g++:

2. Run the compiled executable:

3. Follow the on-screen prompts to interact with the system:
- Enter 1 to add a new order.
- Enter 2 to print a bill for a specific order.
- Enter 3 to mark an order as completed.
- Enter 4 to check for pending orders.
- Enter 5 to display order details by order number.
- Enter 6 to quit the program.

### Dependencies

- None

### Description
